# Dynamic CORE+

## Overview
Dynamic CORE+ is a powerful, all-in-one Minecraft SMP essential plugin designed to provide server owners with everything needed for smooth server management and enriched player experience. It integrates a full suite of vital features found in EssentialsX and enhances them with additional moderation, utility, and quality of life tools, while outsourcing economy functionalities through Vault API integration.

## Core Features

- Essentials: Homes, spawn points, warp system, teleport requests (TPA), random teleport (RTP), back command, AFK detection, multi-world support, and more.

- Economy Integration: Vault-powered balance display, player payments, daily rewards, and compatibility with major economy plugins.

- Moderation & Admin Tools: Mute, ban, kick, freeze, god mode, inventory clearing, player data management, chat controls, whitelist/blacklist, and staff chat.

- Utility Commands: Item repair, renaming, enchantments, player statistics, leaderboards, and light grief protection.

- Quality of Life Improvements: Customizable messages, permission integration, world border management, opt-in metrics, and comprehensive config options.

## Configuration
Dynamic CORE+ features an intuitive, detailed configuration file divided into:

- Essentials Settings

- Economy Integration

- Moderation and Admin Controls

- Utility Features

- Quality of Life Enhancements
Each section allows fine-tuning of every aspect to perfectly tailor the plugin to your server's needs.

## Why Choose Dynamic CORE+?
Designed as a complete and optimized core plugin, Dynamic CORE+ reduces plugin clutter and dependency issues. It empowers server administrators with robust control mechanisms and players with convenience commands, all backed by Vault's reliable economy system.

Whether you run a small SMP or a large multiplayer server, Dynamic CORE+ is built to scale and evolve with community contributions and ongoing updates, making it the definitive choice for your Minecraft server essentials.
